Tmux Resurrect mkdir ~/.tmux; cd ~/.tmux; git clone https://github.com/tmux-plugins/tmux-resurrect.git
run-shell ~/.tmux/tmux-resurrect/resurrect.tmux 添加到~/.tmux.conf ; tmux source-file ~/.tmux.conf
前缀键 + Ctrl-s   # 保存
前缀键 + Ctrl-r   # 恢复
