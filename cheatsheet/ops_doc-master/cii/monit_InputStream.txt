InputStream: 被用来读取binary或者text类型文件
当 底层字符串出现错误，读写数据超时或者EOF时，InputStream_isClosed()返回真
当 InputStream_setTimeout(timeout) timeout>0时，用来处理non-blocking 模式 

T InputStream_new(int descriptor);
void InputStream_free(T *S);

int InputStream_isClosed(T S);
int InputStream_getDescriptor(T S) 

void InputStream_setTimeout(T S, time_t timeout) 
time_t InputStream_getTimeout(T S) 

int InputStream_buffered(T S) 


int InputStream_read(T S)                           # 每次返回一个字节
char *InputStream_readLine(T S, char *s, int size)  # 返回一行
int InputStream_readBytes(T S, void *b, int size)   # 返回指定长度

void InputStream_clear(T S)
