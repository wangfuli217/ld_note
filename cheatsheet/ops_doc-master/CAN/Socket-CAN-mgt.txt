    在 Linux 系统中， CAN 总线接口设备作为网络设备被系统进行统一管理。
在控制台下， CAN 总线的配置和以太网的配置使用相同的命令。
ifconfig -a
# eth0 设备为以太网接口， can0和can1 设备为两个 CAN 总线接口。

使用 ip 命令来配置 CAN 总线的位速率
ip link set can0 type cantq 125 prop-seg 6phase-seg1 7 phase-seg2 2 sjw 1

ip 命令直接设定位速率：
ip link set can0 type can bitrate 125000

当设置完成后，可以通过下面的命令查询 can0 设备的参数设置：
ip -details link show can0

以使用下面的命令使能 can0 设备：
ifconfig can0 up

使用下面的命令取消 can0 设备使能：
ifconfig can0 down

在设备工作中，可以使用下面的命令来查询工作状态：
ip -details -statistics link show can0
