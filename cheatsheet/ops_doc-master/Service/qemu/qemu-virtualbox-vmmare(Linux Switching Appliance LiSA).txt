LiSA is a network multilayer switching device that is be able to perform Layer 2 and Layer 3 packet switching. It should provide a cost effective advanced-featured solution for small networks. The configuration and operation command line interface is similar to Cisco's IOS.

1. LiSA Version 2.0.1 installed on CentOS 6.3
