## run pytest
* `python -m pytest test_um_pytest.py`
* `py.test test_um_pytest.py`

### with verbose
* `python -m pytest -v test_um_pytest.py`
* `py.test -v test_um_pytest.py`

### short version
* `py.test -q test_um_pytest.py`

## run unittest
* `py.test test_um_unittest.py`
