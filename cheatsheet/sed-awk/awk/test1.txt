#员工ID, 姓名, 薪资率,及 实际工时. ID中的第一码为部门识别码. "A","P"分别表示"组装"及"包装"部门.

----- [ 范例 :] 以文件 emp.dat 为例, 计算每人应发工资并打印报表. ----
awk '{ print $2, $3 * $4 }' data1

== print 的参数间彼此以 "," (逗号) 隔开, 印出数据时彼此间会以空白隔开. 

awk '{ printf("%6s Work hours: %3d Pay: %5d\n", $2,$3, $3*$4 ) }' data1

== printf() 函数可进一步控制数据的输出格式. 

----- 打印新的员工薪资率报表. -----
awk '$1 ~ /^A.*/ { $3 *= 1.05 } $3<100 { $3 = 100 } { printf("%s %8s %d\n", $1, $2, $3)}' data1
